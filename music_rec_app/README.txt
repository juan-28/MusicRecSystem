# Music Recommendation System (Music_Rec_v2)

## Setup Instructions

Make sure you have the latest version of pyspark installed on your machine

### Step 1: Unzip the File

1. Locate the `Music_Rec_v2.zip` file on your machine.
2. Unzip the file into a directory of your choice

### Step 2: Navigate to the Uncompressed Directory

1. Open your command-line tool.
2. Use the `cd` (change directory) command to navigate to the directory where you unzipped `Music_Rec_v2.zip`. 


### Step 3: Run the Python Script

1. Once you are in the correct directory, run the `app_test.py` script using Python. Enter the following command:
   ```
   python app_test.py
   ```
2. The script should now execute. Follow any on-screen prompts or instructions that appear to get listened songs and new song recommendations. (It should ask for user ID and you can enter any number to get their users listening history and recommendations based on other users listening data)
